# Extract File

untuk mengextract file, ketikan perintah seperti dibawah ini:

`steghide extract -sf <file hasil steghide> -xf <file untuk tempat hasil ekstrak> -p <password>`

Keterangan

- `-sf = Steghide file`
- `-xf = file yang menjadi target menyimpan hasil ekstrak`
- `-p = password`
